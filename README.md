Blog Generator using Hugo
====

## Install
In terminal, execute the following commands.

`brew install hugo`

`hugo new site blog`


## Theme

### Install themes

`git clone --recursive https://github.com/spf13/hugoThemes.git themes`

### Preview themes

`hugo server -t "`ls themes | peco`"`


## Preview

`hugo server`


## Build

`hugo`


## Deploy

`./deploy.sh`
