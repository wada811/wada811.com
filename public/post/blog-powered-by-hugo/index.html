Hugoでブログを書くことにしました

今まで Blogger でブログを書いていましたが Hugo に移行しました。(過去記事は移行できていないけど…)

移行理由は、主に以下の2つ。

  • Blogger のプレビューが重くて少し書いては1分くらいプレビューが表示されるのを待つという状況
  • 最近は Qiita で Markdown を使って書くようになり、ブログも Markdown で書きたいという欲求

Hugo は Markdown で書いてすぐプレビューができるので最高っぽい。
